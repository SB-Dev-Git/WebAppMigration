# 画面設計

## 概要

勤怠入力アプリケーションの画面設計仕様書です。全5画面の詳細なレイアウト、UI要素、画面遷移、ユーザーインタラクションを定義しています。

## 画面構成

```
勤怠入力アプリケーション
├── ホーム画面 (/)
├── 勤怠入力画面 (/input)
├── 勤怠一覧画面 (/display)
├── CSV取込画面 (/import_csv)
└── CSV出力機能 (/export_csv)
```

## 共通レイアウト (base.html)

### ヘッダー部分
```
┌─────────────────────────────────────────────────────────────┐
│ 勤怠入力アプリケーション    [ホーム][勤怠入力][勤怠一覧][CSV取込][CSV出力] │
└─────────────────────────────────────────────────────────────┘
```

#### 構成要素
- **アプリケーションタイトル**: 左側に配置、フォントサイズ1.5rem
- **ナビゲーションメニュー**: 右側に水平配置、5つのリンク
- **背景色**: #2c3e50（ダークブルー）
- **文字色**: 白色
- **ホバー効果**: 背景色が#34495eに変化

#### レスポンシブ対応
- **768px以下**: ナビゲーションが縦並びに変更
- **メニュー項目**: 中央揃えで折り返し表示

### フラッシュメッセージ

┌─────────────────────────────────────────────────────────────┐
│ ✓ 成功メッセージ（緑色背景）                                      │
│ ✗ エラーメッセージ（赤色背景）                                    │

#### スタイル仕様
- **成功メッセージ**: 背景色#d4edda、文字色#155724
- **エラーメッセージ**: 背景色#f8d7da、文字色#721c24
- **パディング**: 1rem

## 2. 勤怠入力画面 (input.html)

### 画面レイアウト
```
┌─────────────────────────────────────────────────────────────┐
│                        勤怠入力                              │
│                                                             │
│  ┌─────────────────────────────────────────────────────────┐  │
│  │ 日付: [2025-06-30        ▼]                             │  │
│  │                                                         │  │
│  │ 種別: [フレックス         ▼]                             │  │
│  │                                                         │  │
│  │ 出勤時間: [09:00]                                        │  │
│  │                                                         │  │
│  │ 退勤時間: [18:00]                                        │  │
│  │                                                         │  │
│  │ 休憩時間（分）: [60]                                      │  │
│  │                                                         │  │
│  │ コメント:                                                │  │
│  │ ┌─────────────────────────────────────────────────────┐    │  │
│  │ │                                                     │    │  │
│  │ │                                                     │    │  │
│  │ │                                                     │    │  │
│  │ └─────────────────────────────────────────────────────┘    │  │
│  │                                                         │  │
│  │              [保存]  [一覧へ戻る]                         │  │
│  └─────────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
```

### フォーム要素詳細

#### 日付入力
- **タイプ**: date
- **デフォルト値**: 今日の日付
- **必須項目**: ✓
- **JavaScript機能**: 土日選択時に種別を「-」に自動設定

#### 種別選択
- **タイプ**: select（ドロップダウン）
- **選択肢**:
  - 「-」（土日デフォルト）
  - 「フレックス」
  - 「在宅勤務」
  - 「午前休」
  - 「午後休」
  - 「休暇」
  - 「祝祭日」
- **必須項目**: ✓

#### 時間入力フィールド
- **出勤時間・退勤時間**: type="time"
- **フォーマット**: HH:MM（24時間形式）
- **任意項目**: 種別が「-」「休暇」「祝祭日」の場合

#### 休憩時間入力
- **タイプ**: number
- **単位**: 分
- **最小値**: 0
- **デフォルト値**: 0

#### コメント入力
- **タイプ**: textarea
- **行数**: 3行
- **任意項目**: ✓

### バリデーション機能

#### クライアントサイド
- **土日判定**: 日付変更時に種別を自動設定
- **必須項目チェック**: 日付、種別

#### サーバーサイド
- **時間整合性**: 出勤時間 < 退勤時間
- **休憩時間**: 休憩時間 < 総勤務時間
- **時間形式**: HH:MM形式の検証
- **数値範囲**: 休憩時間 >= 0

### ボタン配置
- **保存ボタン**: プライマリスタイル（青色）
- **一覧へ戻るボタン**: セカンダリスタイル（グレー）
- **配置**: 左右に並列配置

### エラー表示
- **フラッシュメッセージ**: フォーム上部に表示
- **エラー内容**: バリデーション失敗時の詳細メッセージ

- **文字色**: #2c3e50
- **パディング**: 1rem

#### データ行
- **ホバー効果**: 背景色#f8f9faに変化
- **ボーダー**: 下線1px solid #eee
- **パディング**: 1rem
- **テキスト整列**: 左揃え

#### 列構成
1. **日付**: YYYY-MM-DD形式
2. **曜日**: 漢字1文字（月火水木金土日）
3. **種別**: 勤務種別

5. **退勤時間**: HH:MM形式（未入力時「-」）
6. **休憩時間**: 「XX分」形式
7. **実働時間**: 「X.X時間」形式
8. **コメント**: テキスト（未入力時「-」）

### セクション構成

#### 説明セクション
- **タイトル**: 「CSVファイル形式について」
- **色**: #2c3e50
- **マージン下**: 1rem
- **リスト項目**: 左マージン1.5rem、色#666

#### ファイル選択フォーム
- **ファイル入力**: type="file", accept=".csv"
- **必須項目**: ✓
- **エンコードタイプ**: multipart/form-data

#### サンプルCSVセクション
- **背景色**: #f8f9fa
- **ボーダー**: 1px solid #e9ecef
- **フォント**: 'Courier New', monospace
- **フォントサイズ**: 0.9rem
- **パディング**: 1rem
- **横スクロール**: 対応

### バリデーション

#### ファイル検証
- **拡張子**: .csvのみ許可
- **ファイル存在**: 必須
- **ファイルサイズ**: 制限なし（推奨10MB以下）

#### データ検証
- **ヘッダー行**: 必須
- **列数**: 8列固定
- **日付形式**: YYYY-MM-DD
- **時間形式**: HH:MM
- **数値形式**: 休憩時間（整数）、実働時間（小数点可）

### 処理フロー
1. ファイル選択
2. 取込実行ボタンクリック
3. サーバーサイド処理
4. 結果表示（成功/エラー）
5. 一覧画面へリダイレクト

## 5. CSV出力機能 (/export_csv)

### 機能概要
- **直接ダウンロード**: 画面表示なし
- **ファイル名**: kintai_YYYYMMDD.csv（実行日付）
- **文字エンコード**: UTF-8 with BOM
- **データ順序**: 日付昇順

### CSV形式
```
日付,曜日,種別,出勤時間,退勤時間,休憩時間,実働時間,コメント
2025-06-25,水,フレックス,09:00,18:00,60,8.0,テスト勤怠データ
2025-06-26,木,在宅勤務,09:30,17:30,60,7.0,在宅勤務
```
