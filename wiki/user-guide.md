# 使用方法

## 基本操作

### ログイン・アクセス

1. ブラウザで以下のURLにアクセス
   - ローカル環境: http://localhost:5000
   - EC2環境: http://your-ec2-ip:5000

2. ホーム画面が表示されます

## 機能別操作マニュアル

### 1. 勤怠入力

#### 基本的な入力手順
1. ナビゲーションメニューから「勤怠入力」をクリック
2. 各項目を入力：
   - **日付**: カレンダーから選択（デフォルト：今日）
   - **種別**: プルダウンから選択
   - **出勤時間**: HH:MM形式で入力（例：09:00）
   - **退勤時間**: HH:MM形式で入力（例：18:00）
   - **休憩時間**: 分単位で入力（例：60）
   - **コメント**: 任意のメモ（例：会議多め）
3. 「保存」ボタンをクリック

#### 勤怠種別の選択

| 種別 | 使用場面 | 備考 |
|------|----------|------|
| - | 土日・祝日 | 出退勤時間入力不要 |
| フレックス | 通常勤務 | 標準的な勤務形態 |
| 在宅勤務 | リモートワーク | テレワーク時に選択 |
| 午前休 | 午前半休 | 午後から出勤 |
| 午後休 | 午後半休 | 午前のみ勤務 |
| 休暇 | 有給休暇等 | 終日休暇 |
| 祝祭日 | 国民の祝日 | 法定休日 |

#### 入力時の注意点
- 出勤時間は退勤時間より前である必要があります
- 休憩時間は総勤務時間より短い必要があります
- 時間は24時間制で入力してください（例：14:30）
- 同じ日付のデータは上書きされます

### 2. 勤怠一覧表示

#### 一覧画面の見方
1. ナビゲーションメニューから「勤怠一覧」をクリック
2. 表形式で勤怠データが表示されます

#### 表示項目
- **日付**: 勤怠日（YYYY-MM-DD形式）
- **曜日**: 月〜日の表示
- **種別**: 勤怠種別
- **出勤時間**: HH:MM形式
- **退勤時間**: HH:MM形式
- **休憩時間**: 分単位
- **実働時間**: 自動計算された時間数
- **コメント**: 入力されたメモ

#### 実働時間の計算
```
実働時間 = 退勤時間 - 出勤時間 - 休憩時間
```

例：
- 出勤：09:00
- 退勤：18:00
- 休憩：60分
- 実働時間：8.0時間

### 3. CSV出力機能

#### CSV出力手順
1. 勤怠一覧画面で「CSV出力」ボタンをクリック
2. ファイルが自動ダウンロードされます
3. ファイル名：`kintai_YYYYMMDD.csv`（実行日付）

#### CSV形式
```csv
日付,曜日,種別,出勤時間,退勤時間,休憩時間,実働時間,コメント
2025-06-25,水,フレックス,09:00,18:00,60,8.0,テスト勤怠データ
2025-06-26,木,在宅勤務,09:30,17:30,45,7.25,リモートワーク
```

#### Excel での開き方
1. Excelを起動
2. 「データ」タブ → 「テキストまたはCSVから」
3. ダウンロードしたCSVファイルを選択
4. 文字エンコード：UTF-8を選択
5. 「読み込み」をクリック

### 4. CSV取込機能

#### CSV取込手順
1. ナビゲーションメニューから「CSV取込」をクリック
2. 「ファイルを選択」ボタンをクリック
3. 取込用CSVファイルを選択
4. 「取込実行」ボタンをクリック
5. 取込結果が表示されます

#### 取込用CSVファイルの作成

**必須ヘッダー行**
```csv
日付,曜日,種別,出勤時間,退勤時間,休憩時間,実働時間,コメント
```

**データ行の例**
```csv
2025-07-01,月,フレックス,09:00,18:00,60,8.0,通常勤務
2025-07-02,火,在宅勤務,09:30,17:30,45,7.25,リモートワーク
2025-07-03,水,午前休,13:00,18:00,30,4.5,病院受診
```

#### 取込時の注意点
- ファイル形式：CSV（.csv拡張子）
- 文字エンコード：UTF-8
- ヘッダー行は必須
- 日付形式：YYYY-MM-DD
- 時間形式：HH:MM
- 既存データは上書きされます
- エラーがある行はスキップされます

## 画面操作のコツ

### ナビゲーション
- 画面上部のメニューバーで機能を切り替え
- 「勤怠アプリ」ロゴクリックでホーム画面に戻る

### フォーム入力
- 必須項目は赤いアスタリスク（*）で表示
- エラーメッセージは画面上部に赤色で表示
- 成功メッセージは緑色で表示

### レスポンシブデザイン
- PC、タブレット、スマートフォンに対応
- 画面サイズに応じてレイアウトが自動調整

## データ管理のベストプラクティス

### 日次運用
1. **毎日の入力**: 勤務終了時に当日分を入力
2. **データ確認**: 一覧画面で入力内容を確認
3. **修正**: 間違いがあれば再入力で上書き

### 週次・月次運用
1. **データ確認**: 週末に1週間分のデータを確認
2. **CSV出力**: 月末に月次データをCSVで出力
3. **バックアップ**: 重要なデータは定期的にバックアップ

### データ整合性チェック
- 出勤・退勤時間の論理チェック
- 休憩時間の妥当性確認
- 実働時間の計算結果確認

## よくある質問（FAQ）

### Q1: 同じ日付のデータを複数回入力するとどうなりますか？
A1: 最新の入力内容で上書きされます。1日1レコードの制限があります。

### Q2: 時間を間違えて入力した場合の修正方法は？
A2: 同じ日付で正しい時間を再入力してください。自動的に更新されます。

### Q3: CSVファイルが文字化けします
A3: Excelで開く際に、文字エンコードをUTF-8に設定してください。

### Q4: 休憩時間を入力し忘れた場合は？
A4: デフォルトで0分が設定されます。後から修正可能です。

### Q5: 土日のデータは入力が必要ですか？
A5: 土日は自動的に種別「-」が設定されます。出勤した場合のみ入力してください。

### Q6: 過去のデータを一括で入力したい場合は？
A6: CSV取込機能を使用して、過去データを一括登録できます。

### Q7: データを削除したい場合は？
A7: 現在の画面では削除機能はありません。空のデータで上書きしてください。

### Q8: 実働時間の計算が合わない場合は？
A8: 出勤・退勤・休憩時間を確認し、24時間制で正しく入力されているか確認してください。

## トラブル時の対処

### エラーメッセージが表示された場合
1. エラー内容を確認
2. 入力値を修正
3. 再度保存を実行

### 画面が表示されない場合
1. URLが正しいか確認
2. アプリケーションが起動しているか確認
3. ネットワーク接続を確認

### データが保存されない場合
1. 必須項目が入力されているか確認
2. 時間形式が正しいか確認
3. ブラウザを再読み込み

詳細なトラブルシューティングは[トラブルシューティング](Troubleshooting)ページを参照してください。
