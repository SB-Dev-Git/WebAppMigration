# アーキテクチャ

## システム構成

勤怠入力アプリケーションは、シンプルな3層アーキテクチャを採用しています。

```
┌─────────────────┐
│  プレゼンテーション層  │  ← HTML/CSS/JavaScript
├─────────────────┤
│   アプリケーション層   │  ← Flask (Python)
├─────────────────┤
│    データ層      │  ← SQLite Database
└─────────────────┘
```

## 技術スタック

### バックエンド
- **Flask 2.2.5**: Webアプリケーションフレームワーク
- **Werkzeug 2.2.3**: WSGIユーティリティライブラリ
- **SQLite**: 軽量データベース
- **Python 3.7.16**: プログラミング言語

### フロントエンド
- **HTML5**: マークアップ言語
- **CSS3**: スタイルシート（レスポンシブデザイン）
- **JavaScript**: クライアントサイドスクリプト
- **Jinja2**: テンプレートエンジン

### 開発・運用
- **Git**: バージョン管理
- **GitHub**: リポジトリホスティング
- **AWS EC2**: 本番環境（Amazon Linux 2）

## アプリケーション構造

```
WebAppMigration/

```

## コンポーネント詳細

### 1. Flaskアプリケーション (app.py)

#### 主要関数
- `init_db()`: データベース初期化
- `get_day_of_week()`: 曜日計算
- `calculate_work_hours()`: 実働時間計算
- `validate_time_input()`: 入力バリデーション

#### ルート定義
- `/`: ホーム画面
- `/input`: 勤怠入力
- `/display`: 勤怠一覧
- `/export_csv`: CSV出力
- `/import_csv`: CSV取込

### 2. データベース層

#### 接続管理
- SQLite3を使用した軽量データベース
- 接続プールなし（シンプルな接続管理）
- トランザクション管理

#### データ操作
- INSERT OR REPLACE: 重複データの更新
- SELECT: データ取得（日付順ソート）
- バリデーション: データ整合性チェック

### 3. テンプレート層

#### ベーステンプレート (base.html)
- 共通レイアウト定義
- ナビゲーションメニュー
- フラッシュメッセージ表示
- レスポンシブデザイン

#### 個別テンプレート
- **index.html**: 機能概要とナビゲーション
- **input.html**: 勤怠入力フォーム
- **display.html**: データ一覧表示
- **import.html**: CSVファイルアップロード

## データフロー

### 1. 勤怠入力フロー
```
ユーザー入力 → バリデーション → データベース保存 → 確認画面
```

### 2. 勤怠表示フロー
```
データベース取得 → テンプレート描画 → HTML出力
```

### 3. CSV出力フロー
```
データベース取得 → CSV形式変換 → ファイルダウンロード
```

### 4. CSV取込フロー
```
ファイルアップロード → 形式検証 → データベース保存 → 結果表示
```

## セキュリティ考慮事項

### 入力検証
- 時間形式バリデーション（HH:MM）
- 時間整合性チェック（出勤 < 退勤）
- ファイル形式検証（CSV拡張子）

### データ保護
- SQLインジェクション対策（パラメータ化クエリ）
- ファイル名サニタイゼーション
- エラーハンドリング

### 制限事項
- 認証機能なし（将来実装予定）
- HTTPS未対応（本番環境で要設定）
- セッション管理簡易実装

## パフォーマンス特性

### データベース
- SQLite: 軽量、ファイルベース
- 同時接続制限: 読み取り並行、書き込み排他
- 適用規模: 小〜中規模（〜1000ユーザー）

### メモリ使用量
- Flask: 軽量フレームワーク
- SQLite: インメモリキャッシュ
- 推奨メモリ: 512MB以上

### レスポンス時間
- 画面表示: <100ms
- CSV出力: データ量に依存
- CSV取込: ファイルサイズに依存
