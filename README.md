# 勤怠入力アプリケーション

Flask + SQLiteを使用した勤怠管理Webアプリケーションです。

## 機能

- **勤怠入力**: 日付、種別、出退勤時間、休憩時間、コメントの入力
- **勤怠一覧**: 入力データの表形式表示（実働時間自動計算）
- **CSV出力**: 勤怠データのCSVエクスポート（kintai_YYYYMMDD.csv形式）
- **CSV取込**: CSVファイルからの一括データインポート

## セットアップ手順

### 1. 仮想環境の作成と有効化
```bash
python3 -m venv venv
source venv/bin/activate
```

### 2. 依存関係のインストール
```bash
pip install -r requirements.txt
```

### 3. アプリケーションの起動
```bash
python app.py
```

### 4. ブラウザでアクセス
アプリケーションが起動したら、ブラウザでアクセスしてください。

## 使用方法

### 勤怠入力
1. ナビゲーションメニューから「勤怠入力」を選択
2. 日付、種別（フレックス、在宅勤務等）、出退勤時間、休憩時間、コメントを入力
3. 「保存」ボタンをクリック

### 勤怠一覧表示
- ナビゲーションメニューから「勤怠一覧」を選択
- 入力済みデータが表形式で表示されます
- 実働時間は自動計算されます（退勤時間 - 出勤時間 - 休憩時間）

### CSV出力
- 勤怠一覧画面で「CSV出力」ボタンをクリック
- kintai_YYYYMMDD.csv形式でダウンロードされます

### CSV取込
1. ナビゲーションメニューから「CSV取込」を選択
2. 指定形式のCSVファイルを選択
3. 「取込実行」ボタンをクリック

## CSVファイル形式

```csv
日付,曜日,種別,出勤時間,退勤時間,休憩時間,実働時間,コメント
2025-01-01,水,祝祭日,,,0,0,元日
2025-01-02,木,フレックス,09:00,18:00,60,8.0,通常勤務
```

## 技術仕様

- **言語**: Python 3.13
- **フレームワーク**: Flask
- **データベース**: SQLite
- **フロントエンド**: HTML5, CSS3, JavaScript
- **コーディング規約**: PEP8準拠

## バリデーション機能

- 出勤時間 < 退勤時間のチェック
- 休憩時間 < 勤務時間のチェック
- 時間形式（HH:MM）の検証
- CSVファイル形式の検証

## 種別選択肢

- `-` (デフォルト、土日)
- `フレックス`
- `在宅勤務`
- `午前休`
- `午後休`
- `休暇`
- `祝祭日`
